Image analysis

Software for super-resolution fluorescence microscopy

Software has been crucial to super-resolution microscopy since the beginning of the field, and we have a longstanding interest in pushing the limits of single molecule methods, especially STORM/ PALM super-resolution microscopy and single molecule tracking, by developing new single molecule image analysis tools.

Current/ recent software

We host our code on GitHub, you can find it here: https://github.com/HoldenLab

Older software

  • AutoLase (Micromanager plugin), Automatic closed loop control of density of photoactive molecules during PALM experiments.
  • DAOSTORM, multi-emitter fitting for STORM/ PALM.
  • TwoTone, single molecule FRET data analysis software.
